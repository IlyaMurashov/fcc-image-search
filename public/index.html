Image search microservice

A freeCodeCamp project by Ilya Murashov

This service allows you to find images using Google Custom Search API. The search result is return in form of JSON array containing JSON objects with link, alt, and foundOn fields representing the image link, image description, and the website address the image was found on, respectively.

Usage example

The entry point for the image search is /api/[string to query]?offset=[pagination]. If offset query is omitted from the URL, the first page will be returned.

https://fcc-im-image-search.herokuapp.com/api/dank%20memes
or
https://fcc-im-image-search.herokuapp.com/api/dank%20memes?offset=2

A JSON array with 10 results representing a search page will be returned.

Last queries

To the latest 10 queries performed using the service navigate to /latest/ which returns a JSON array of 10 latest queries.
